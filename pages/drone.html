Vision-Guided Quadrotor Drone

Summer - Fall 2023

Project Description

The goal of this project was to enable a drone to autonomously follow a ground robot. I achieved this by collecting and annotating a custom dataset of images of a Jackal UGV, then training a YOLOv5 object detection model. Next, I developed an object-tracking control pipeline that converted real-time model outputs into drone flight commands. I tested using a Tello drone, and I iterated until the drone could continue tracking a target object that briefly left the frame.
